# 효자고 2학년 문학 — 학기말 심화탐구 주제 찾기 (정적 웹앱)

학생이 인상 깊었던 작품(1~3개)을 고르면, 그 작품을 ‘씨앗’으로 삼아 **문학 고유의 심화탐구 주제**를 유형별로 추천하는 단일 페이지 웹앱.

## 특징
- **순수 정적 HTML 1개** (`index.html`) — 서버·API·키 불필요, 비용 0, 즉시 응답
- 주제는 사전 생성 + **적대적 검증 레이어**를 통과한 뱅크를 내장 (환각 0, 교사 검수 가능)
- 진로 연계는 본체가 아니라 ‘향’ 정도(주제별 선택적 `linkHint` 한 줄)

## 탐구 유형 10종
주제·정서 확장 / 소재·배경 확장 / 갈래·문학사 흐름 / 작가 심화 / 매체 확장 /
표현·형식 탐구 / 작품·작가 비교 / 창작·재구성 / 비평·해석 쟁점 / 시대·창작 배경

## 주제 뱅크 수정
`index.html` 안의 `const BANK = [...]` 배열을 고치면 됩니다.
각 항목: `{ type, title, seeds[], activity, linkHint, skeleton{question, steps[3], output} }`

## 배포
GitHub Pages 등 정적 호스팅에 `index.html` 하나만 올리면 끝.
